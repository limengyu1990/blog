Network & Web Programming

HTTP

Haskell具有各种HTTP请求和处理库。最简单，最灵活的是HTTP库.

Blaze

Blaze是HTML组合器库，它提供了以编程方式构建HTML的可组合位的能力。

它没有像Hastache这样的字符串模板库，而是提供了一个API，用于从逻辑上构建HTML文档，在逻辑上，程序会生成输出的格式。

为了对HTML元素进行排序，这些元素可以按monad或使用monoid操作进行排序.

对于自定义数据类型，我们可以实现ToMarkupclass以在Haskell数据结构和HTML表示之间进行转换.

Warp

Warp是一种高效的Web服务器，它是几种流行的Haskell Web框架背后的后端请求引擎。内部已经过微调，可以利用Haskell的并发运行时，并且能够处理大量并发请求.

Scotty

继续我们在网络库中的跋涉，Scotty是一个网络微框架，其原理与Python中的Flask或Ruby中的Sinatra相似。

需要注意的重要一点是，这里使用的Blaze库重载了do-notation，但它本身并不是proper(适当的)monad，因此通常适用于monad的各种定律和不变量可能会因错误而失效或失败。

Hastache

Hastache是基于Mustache样式的字符串模板，该样式使用双括号编码元变量。

Hastache支持将许多Haskell类型自动转换为字符串，并使用高效的Text函数进行格式化。

在将变量名称映射到可打印的MuType值的函数中，指定了加载到模板中的变量。例如使用函數。

或使用Data-Typeable记录和mkGenericContext，将Haskell字段名称转换为变量名称.

MuType和MuContext类型可以由实现MonadIO的任何monad或转换器(而不仅仅是IO)进行参数化.
